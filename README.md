# DeutschEcho_Stadtrettung
Die Stadt ist still. Wörter verschwinden. Stimmen verstummen. Nur du kannst sie zurückholen.  Laufe durch die Straßen, entschlüssele Hinweise, rette verlorene Stimmen und bring Deutsch zurück ins Leben. Mit jedem Level wird die Welt komplexer – und dein Deutsch stärker. Das ist kein Spiel zum Lernen. Das ist ein Spiel zum Überleben… auf Deutsch.
